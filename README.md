Vi startet prosjektet med å sette oss inn i react og få ein viss oversikt ov verktøyene vi hadde tilgjengelig. Deretter bestemte vi oss for å dele oss opp i 3 for å få mest mogleggjort. ein skulle lære seg tekst rendering, ein svg rendering og den siste audio track bytte. i tillegg starta den eine å lage eit rammeverk i CSS og rendre HTML
slik at en kunne bare legg til funksjonalitet senere.  Valgte her å benytte CSS-grid og CSS-flexbox sammen for å lage vår eigen verson av layouten brukt som eksempel i oppgåve teksten. Dette fordi vi syns den var oversiktlig nok til dette prosjektet og den er enkel å lage. I tillegg gjer den det enkelt å plassere innhold i kvar sin «boks» når funksjonaliteten kom seinare. Vi bestemte oss for å benytte knapper for å sende informasjon om kva tab du er i. for at det skal bli oversikktlig for brukeren bytter knappen farge til svart på den tabben som er valgt. I kollonna til høgre lagde vi tre forms med radiobuttons for å velje tema for tabsa. Eit form per tema. Vi valgte denne måten sidan det er vanleg
og forståeleg for brukaren at radiobuttons held variable over lengre tid. 

Materialet som blir/skulle blitt vist er for det meste henta frå nettet. Audio frå: bensound.com. Svg frå: publicdomaincvectors.org og teksten er henta litt frå rundt om på nettet.

Det var krav i oppgåven å ha responsivt Webdesign. Med krav om veiwport,media-queries,skalering av bilder og flexibel layout.
